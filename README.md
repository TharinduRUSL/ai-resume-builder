# AI Resume Builder

A modern Resume Builder application built with React JS and Vite.

## Features

* Live Resume Preview
* Profile Photo Upload
* GitHub & LinkedIn Links
* Experience Section
* Projects Section
* Technical Skills Selection
* PDF Resume Download
* Responsive Design
* Modern UI

## Technologies Used

* React JS
* Vite
* JavaScript
* CSS3
* jsPDF
* React Icons

```bash
git clone https://github.com/TharinduRUSL/ai-resume-builder.git

cd ai-resume-builder

npm install

npm run dev
```
